# Pre-v0 documentation (archive)

This directory contains the project's documentation from before the v0 design reframe of 2026-05-25.

**These files are NOT applicable to the current architecture.** They are kept only for historical reference. For the current authoritative docs, see `PROBLEM.md`, `DESIGN.md`, and `DECISIONS.md` in the project root.

Contents:

- `PROBLEM.md` — pre-reframe problem statement.
- `NEXT.md` — pre-reframe direction-setting doc, captured the "precision-first landings with built-in variety" milestone that the reframe superseded.
- `design-draft-2026-05.md` — first draft of the v0 design, captured during the reframe conversation and reviewed by sub-agents before the polished `DESIGN.md` replaced it.
- `DISCUSSION-2026-05.md` — curated summary of the design conversation that was used to brief the sub-agent reviewers.

Note: there was no top-level `DESIGN.md` before this reframe; the design lived in `DESIGN-DEPRECATED.md` and `NEXT.md` informally.
